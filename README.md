## About The Project

This project is a real-time 2D flight simulation where you interact with air traffic control using voice commands.

You must follow instructions, manage your aircraft, and land safely while avoiding other traffic.

The goal is to combine gameplay with practical voice recognition systems in a clean and modular architecture.

---

## Built With

* C++ (SDL2)
* Python (Speech-to-Text, Text-to-Speech)
